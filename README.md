# 🚀 DevSync AI

An AI-powered Project Management application built with **React, Firebase, Firestore, React Router, and Vite**.

🌐 **Live Demo:** https://dev-sync-ai-five.vercel.app

---

## 📌 Features

- 🔐 Firebase Authentication (Login & Signup)
- 👤 User-specific Dashboard
- 📁 Create, Edit & Delete Projects
- ✅ Create, Update & Delete Tasks
- 📊 Dashboard Statistics
- 📈 Dynamic Progress Bar
- 📝 Recent Projects Section
- 🤖 AI Reviews Page
- ⚙️ Settings Page
- ☁️ Cloud Firestore Database
- 🌍 Deployed on Vercel

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/abhishekkrsri/DevSync-AI.git
```

Go to the project folder:

```bash
cd DevSync-AI/frontend
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
